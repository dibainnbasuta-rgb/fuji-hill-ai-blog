外ライド・実走2026.06.23

太平山317Wより、
帰りのストレート280Wが収穫だった日

今日は太平方面へ外ライド。走り始めはあまり踏めない感じだった。昨日の唐沢＋琴平がTSS133.7だったので、当然といえば当然である。脚に「昨日も結構やりましたよね？」と確認されている感じがした。

ただ、名もなき峠を越えたあたりから体が動き始めた。そこからは下皆川街道、太平山、帰りのストレートまで、思ったよりしっかり踏めた。今日の収穫は太平山317Wもそうだが、個人的には帰りのストレート16分32秒280Wの方が大きかった。

全部読むのが面倒な人向けまとめ

  • 太平方面へ外ライド。走り始めは踏めなかったが、名もなき峠のあとから調子が上がった。
  • 太平山は7分17秒317W。数字としてはかなり良い。
  • 一番の収穫は帰りのストレート16分32秒280W。無理に踏んでいないのに力が逃げない感覚があった。
  • 全体は40.65km、NP253W、IF0.918、TSS119.5。直近4回の練習負荷はかなり高い。
  • Aethos2のサドルまわりは少し気になる。強く踏めるようになったからこそ、機材との相性も見えてきた。

今日の主な区間

今日のライド全体は40.65km、1時間25分28秒、獲得標高481m。平均パワー199W、NP253W、IF0.918、TSS119.5だった。軽い流しではない。昨日もかなり踏んでいるのに、今日も普通に負荷が高い。少し自分に確認したい。休むという概念はどこに置いてきたのか。

主な区間はこの5つ。今後、太平方面のラップは名もなき峠、下皆川街道、下皆川街道上り返し、太平山、帰りのストレートとして見ていく。

  • 名もなき峠2分23秒 / 318W / NP314W
  • 下皆川街道2分42秒 / 349W / NP349W
  • 下皆川街道上り返し45秒 / 422W / NP396W
  • 太平山7分17秒 / 317W / NP318W
  • 帰りのストレート16分32秒 / 280W / NP284W

走り始めは微妙だった

走り始めは、正直あまり踏める感じではなかった。昨日は唐沢5分337W、琴平12分305W、ラストストレート16分265W。TSS133.7。普通に考えれば、今日は脚が重くてもおかしくない。

ただ、名もなき峠を越えたあたりで「結構いけるかも」という感覚が出てきた。ここで2分23秒318W。短い区間とはいえ、入りとしては悪くない。そこから下皆川街道で2分42秒349W、上り返しで45秒422W。体が起きてきたというより、急にやる気を出し始めた。そういう日がある。

太平山：7分17秒 317W

太平山は7分17秒で平均317W、NP318W。FTP設定275Wに対して約115％。昨日の疲労を考えると、かなり良い数字だと思う。最大心拍は168bpmまで上がっているので、しっかり追い込んでいる。

ただ、今日の主役を太平山だけにすると少し違う気がした。太平山317Wはもちろん強い。でも今日の感触として一番大きかったのは、登りが終わったあとにもう一度踏めたことだった。

帰りのストレート：16分32秒 280W

今日一番の収穫は、帰りのストレートだった。16分32秒で平均280W、NP284W。前日の唐沢＋琴平ライドでは同じルートが16分57秒265Wだったので、今回は明確に上がっている。しかも、太平山を踏んだあとである。

体感としては、呼吸が整っていた。無理に踏んでいる感じではないのに、力が逃げずにずっと踏めるような感覚があった。もちろん実際にずっと無限に踏めるわけではない。人間なので。だが、今日は「踏む力がペダルにちゃんと乗っている」感じがあった。

最近ローラーでSSTとVO2を続けている。SSTは長く踏む練習、VO2は上限を押し上げる練習。その両方が、外のこの区間で少しつながった気がした。登りの数字も大事だが、帰りのストレートで16分280Wをまとめられたことは、かなり大きい。

ローラーの練習が外につながってきたかもしれない

直近の練習を並べると、SSTローラーがTSS66.7、VO2ローラーがTSS73.6、唐沢＋琴平がTSS133.7、今日の太平がTSS119.5。合計すると、なかなかの負荷である。数字だけ見ると、かなり元気におかしなことをしている。

でも、今日の感覚は悪くなかった。走り始めは微妙でも、途中から踏める。短い登りで上げられる。太平山で317Wを出せる。そのあと帰りのストレートで16分280Wを出せる。これは、ローラーで積んでいるSSTとVO2が少しずつ外でも効いてきた、と見てもよさそうだ。

もちろん、これで完成ではない。1時間300Wはまだ遠い。富士ヒルゴールドも遠い。でも「できないことを一歩ずつ潰す」という意味では、今日もかなり良い一日だった。

サドルまわりも少し気になった

一方で、Aethos2のサドルまわりは少し気になる点が出てきた。今のMirror系サドルは滑りにくく、ダンシング後に座り直す時にポジションを調整しづらい感じがある。最近レーパン直履きに変えたことで、今まで隠れていた相性問題が表に出てきた可能性もある。

アンカーの通常Power系サドルでは問題ない感じなので、Aethos2でもMirrorではないPower系を試す方向はありそうだ。強く踏めるようになってきたからこそ、機材との相性も見えてくる。脚だけ強くなれば全部解決、とはいかない。自転車、面倒である。そこが面白いところでもある。

明日は休む方向でいい

今日はTSS119.5。推定発汗量は1,116ml、運動消費は1,020kcal。ライド中の補給は太平で水を汲んだくらいなので、帰宅後は水分、塩分、炭水化物を戻すのが大事になる。夜はエキストラアミノアシッドも入れておきたい。

直近の負荷を考えると、水曜はレストでいいと思う。というより、ここで休まないと何のためにデータを見ているのか分からなくなる。踏めた日は気分が良い。でも、踏めた日の翌日にちゃんと回復するところまでが練習である。たぶん。

今日やってみて思ったこと

今日は、太平山317Wも良かった。でも、それ以上に帰りのストレート16分32秒280Wが良かった。呼吸が整って、無理に踏んでいないのに力が逃げない。あの感覚はかなり良い。

走り始めは微妙だったのに、途中から体が動いて、最後までしっかり踏めた。ローラーで積んでいるSSTとVO2が、少しずつ外に出てきている気がする。まだ足りない。でも、足りないなりに前へ進んでいる。

できないことを一歩ずつ潰していく途中の、かなり良い一日だった。明日は休む。たぶん休む。ここはAIに怒られる前に休む。
